# 🤖 GenAI Chatbot (CLI + Web UI)

A ChatGPT-like chatbot built using **OpenAI API** and **Streamlit**, supporting interactive conversations and customizable AI behavior.

---

## 🚀 Features

- ✅ Interactive chatbot UI using Streamlit
- ✅ Chat history support
- ✅ Custom system prompt (define AI behavior)
- ✅ Clear chat functionality
- ✅ Modular and clean code structure

---

## 🧠 Tech Stack

- Python
- OpenAI API
- Streamlit
- dotenv
